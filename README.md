#我的OJ日记
## my_onlinejudge_diary

#前言
现在是2022年11月5日，我在学校实验室里的工位上编辑着这个项目，突然想写一段前言，来讲述我高中时代在oi上的付出。

2020年10月，我正值高二。
当时我记得很清楚，我是在知乎上了解到信息学这一方向的竞赛，也在这过程中逐渐对编程这方面的知识有了新的认知。
从小我就是一个爱折腾的人，也是一个一旦有自己的理想就要奋不顾身去实现的人。
于是，我想尝试一下。
那年12月，正值洛谷网的创始人kkksc03的新书《深入浅出程序设计竞赛》出版，我买了一本，经常翻阅。
（我最先买到的是刘汝佳大佬写的《算法竞赛入门经典》，但当时感觉那本书难度不太适合我......）
但是因为高中学业较忙，加上那时我还没电脑，所以我很少上手实操。
在此感谢那时我的父母和高中时期的两位信息课老师，他们在得知我一直在学，也一直在鼓励我去实现我的想法。
